# UC04 - View Subscription Catalog

| Field                | Description |
|----------------------|-------------|
| **Goal**             | Exhibts every subscription present in the database |
| **Actor**            | User        |
| **Pre-conditions**   | Having the application running |
| **Nominal Scenario** | 1. The User prompts the system to exhibt the subscription service catalog<br>2.The system exhibts every subscription service present in the catalog<br>3. The User prompts to quit the catalog view |
| **Post-conditions**  | The User has had access to the subscription service catalog |
| **Exceptions**       | 2a. The system cannot connect to the database: the user is notified and the operation is cancelled. |
